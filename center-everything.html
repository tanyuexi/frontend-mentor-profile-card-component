How to Center Anything in CSS

To center texts horizontally

Parent class

  1. text-align: center;

Target class

  1. display: block; // default

This is p

To center blocks horizontally

Parent class

  1. text-align: center;

Target class

  1. display: block; // default

To center blocks of fixed sizes horizontally

Parent class

Target class

  1. display: block; // default
  2. width: 200px;
  3. margin: auto;

To center anything of a fixed size horizontally and vertically

Parent class

  1. position: relative;

Target class

  1. position: absolute/fixed; //out of layout flow
  2. width: 200px;
  3. height: 100px;
  4. top: 50%;
  5. left: 50%;
  6. margin-top: -50px; //half height
  7. margin-left: -100px; //half width
